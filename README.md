- [x] Interfaz sencilla

- [x] Uso de Browser router basado en la history API html5

## 3. Pre-requisitos

_Necesario `npm`_


## 4. Clone and use 

Clonar el repositorio e instalar dependencias
~~~
  git clone https://github.com/JuanC-JC/CarouselMovies-ReactJs
  npm install 
~~~

Ejecutar en localhost
~~~
  npm run start
~~~

Deploy (bundle) 
~~~
  npm run build
  npm install serve -g 
  serve -s (folder)
~~~
